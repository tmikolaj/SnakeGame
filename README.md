A classic Snake game built using the SFML in C++.

## ⚙️ Technologies

- **C++**
- **SFML** (Graphics, Window, Events)

---

## 🚀 How to Run

1. Install SFML library
2. Clone the repository
3. Build using CMake or your preferred IDE (e.g., CLion)
4. Run the executable
